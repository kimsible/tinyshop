# tinyshop

## Development

#### Install dependencies
```bash
npm install
```

#### Watch tests

```bash
npx ava -v -w <DIRECTORY>
```


#### Lint code

```bash
npx standard -v
```

#### Git hooks

**Husky** formats code and runs tests for staged-files on pre-commit.

#### Start app

##### Development mode
```bash
node server
```

##### Build for production
```bash
npm run build
```

##### Production mode
```bash
npm start
```
