# tinyshop

## Development

#### Install dependencies
```bash
npm install
```

#### Watch tests

```bash
npx ava -v -w
```


#### Lint code

```bash
npx standard -v
```

#### Git hooks

**Husky** formats code on pre-commit and runs tests on pre-push.

#### Start app

##### Development mode
```bash
node server
```

##### Build for production
```bash
npm run build
```

##### Production mode
```bash
npm start
```
